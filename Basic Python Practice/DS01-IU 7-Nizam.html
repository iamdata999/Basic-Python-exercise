#1.Importing libraries:

#numpy (imported as np) is a popular library for numerical computing in Python.
#It provides various functions to generate random numbers and perform mathematical operations efficiently.
#matplotlib.pyplot (imported as plt) is a module within the Matplotlib library that provides a MATLAB-like 
#interface for creating plots and visualizations.

#2.Generating random data:

#np.random.seed(42) sets the seed for the random number 
#generator to ensure reproducibility. It means that every time
#you run the code with the same seed, you'll get the same set of random numbers.
# n = 100 defines the number of data points we want to generate.
# x = np.random.randn(n) generates n random numbers 
# from a standard normal distribution (mean=0, standard deviation=1) for the X values.
# y = np.random.randn(n) generates n random numbers from a standard normal distribution for the Y values.

#3.Creating the scatter plot:

#plt.scatter(x, y) creates a scatter plot by plotting the X 
#and Y values against each other. Each point in the plot represents a pair of X and Y values.

#4.Adding labels and title:

#plt.xlabel('X') sets the label for the X-axis.
#plt.ylabel('Y') sets the label for the Y-axis.
#plt.title('Scatter Plot') sets the title for the plot.

#5.Displaying the plot:

#plt.show() displays the scatter plot on the screen.
#The code uses the randn() function from NumPy to generate
#random data from a standard normal distribution. The scatter plot 
#visually represents the relationship between the X and Y values,
#allowing us to observe any patterns or correlations between the variables.


import numpy as np
import matplotlib.pyplot as plt

# Generate random data
np.random.seed(42)  # Set seed for reproducibility
n = 100  # Number of data points
x = np.random.randn(n)  # Random X values
y = np.random.randn(n)  # Random Y values

# Create scatter plot
plt.scatter(x, y)

# Add labels and title
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Scatter Plot')

# Display the plot
plt.show()

#Task 3: Determine the cumulative profit for all months and display it as a line plot using the following style properties.

#In this code, we have made the following modifications to meet your requirements:

#Line style: linestyle=':' sets the line style to dotted.
#Line color: color='red' sets the line color to red.
#Legend: plt.legend(['Cumulative Profit'], loc='lower right')
#adds a legend with the label 'Cumulative Profit' and positions it in the lower right corner of the plot.

#X-label: plt.xlabel('Month Number') sets the X-axis label to 'Month Number'.
#Y-label: plt.ylabel('Number of Units Sold') sets the Y-axis label to 'Number of Units Sold'.
#Line marker: marker='o' adds a circular marker at each data point.
#Line width: linewidth=3 sets the line width to 3 pixels.

#imported the necessary libraries (matplotlib.pyplot) and that
#the column names in the DataFrame ('Month' and 'Total_profit') match the ones used in the code.


# Calculate the cumulative profit
cumulative_profit = Dataset_mat['Total_profit'].cumsum()

# Set the style properties
plt.plot(Dataset_mat['Month'], cumulative_profit, color='red', linestyle=':', marker='o', linewidth=3)

# Add labels and title
plt.xlabel('Month Number')
plt.ylabel('Number of Units Sold')
plt.title('Cumulative Profit for Each Month')

# Display the legend in the bottom right corner
plt.legend(['Cumulative Profit'], loc='lower right')

# Display the line plot
plt.show()
